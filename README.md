# DyllaOS
Bare Metel OS made to run on the raspberry pi 4

You need to have rasbian already installed on your pi's sd card in able to run this OS.

# Building the OS
You just run the command "make" in the source directory that has the Makefile in it. 

# Installing the OS
Connect the pi's sd card to the computer mount the boot partition. Then copy over and overwirte the config.txt and kernel8.img files.

# Running the OS
Insert the sd card back in the pi, then turn it on. And like magic the DyllaOS boots.
